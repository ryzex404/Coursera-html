HTML ENTITIES

El proposito de esta pagina, es aprender acerca de algunos caracteres que no se pueden introducir debido que son simbolos reservados en HTML. Por lo tanto utilizamos codigos especiales para representar estos caracteres < > &

Para representar el simbolo de © copyright tambien se utiliza un codigo especial.

Al reducir el tamaño de la ventana de una pagina web, el contenido el texto se adapta al tamaño de ventano, causando algunos saltos de linea, de modo que para evitar esto usamos un codigo especial non breaking space nbsp, de igual manera para mostrar comillas utilizamo quot "".
