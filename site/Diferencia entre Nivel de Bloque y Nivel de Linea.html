Diferencia entre Nivel de Bloque y Nivel de Linea

1. Este es un ejemplo de una etiqueta de nivel de bloque:
Inicia una nueva línea, ocupa todo el ancho disponible y puede contener otros elementos de bloque o línea. Algunos ejemplos comunes son: "div", "p", "h1"-"h6", "ul", "li".
2. Por otro lado las etiquetas de nivel de linea: No inicia una nueva línea, ocupa solo el ancho necesario y generalmente no puede contener elementos de bloque, solo elementos de línea o texto.
Algunos ejemplos comunes son: "span", "strong", "em", "a".
