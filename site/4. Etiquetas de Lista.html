Etiquetas de Lista

Esta es una pagina de prueba, para aprender acerca de algunas etiquetas de lista
Etiqueta ul: Significa Unordered List, y sirve para crear listas donde el orden no es relevante
Etiqueta li: Significa List Item, y sirve para representar un elemento individual en una lista
Etiqueta ol: Significa Ordered List, y sirve para crear listas donde el orden importa
