Creando links

El proposito de esta pagina es aprender acerca de algunas etiquetas que nos permiten reedirigirnos a otros sitios dentro o fuera de nuestra pagina.

Links Internos

Etiqueta <a>: Esta etiqueta se utiliza para crear links, para ello se necesita un atributo href, que significa referencia de hipertexto Link a un archivo en el mismo directorio El ejemplo anterior es un link hecho con una etiqueta sin incluir div
La etiqueta <a> es un elemento a nivel de linea y a nivel de bloque, por lo tanto dentro de la etiqueta podemos contener un elemento div, de este modo permitiendonos separar el link del resto del contenido.

Links Externos

El siguiente link nos redirige a una pagina externa a la nuestra Link a mi LinkedIn Dentro de la etiqueta "a" estoy utilizando el atributo target="blank" que basicamente su funcion es, al momento de abrir el link se abre en una pestaña nueva automaticamente, en lugar de abrirse en la pestaña actual.
