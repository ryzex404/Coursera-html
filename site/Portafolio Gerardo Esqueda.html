Acerca de Mi

Esta es una página de prueba diseñada para aprender sobre la anatomía de una regla de CSS. Básicamente, una regla de CSS consta de un selector que elige una etiqueta específica dentro de tu página web. Luego, encontramos las llaves que contienen la declaración del elemento previamente seleccionado. La declaración, a su vez, se compone de dos partes: la propiedad y el valor. Ejemplo: p { color: blue;} En este ejemplo, "p" es el selector que elige todos los párrafos dentro de nuestra página web y nos permite modificar su estilo. Dentro de las llaves, la propiedad es "color" y el valor es "azul". Cabe destacar que para cada propiedad existen valores predefinidos. Para dar estilo a esta página de prueba, he optado por incluir los estilos directamente en el archivo HTML, en lugar de crear un archivo .css como se debería hacer. Utilicé la etiqueta style dentro de la etiqueta head para indicar algunos estilos específicos para el contenido de la página.
